# Draft content

Produce drafts aligned with `AGENT_CORE.md` tone. Prefer outlines → draft → revision prompts.

## Workflow

1. Confirm audience, channel, length, and constraints.
2. Produce draft with clear sections.
3. Leave explicit **Open Questions** when brand voice or facts are unclear.
4. **`remember`** style preferences via `relationship` / `semantic`.
