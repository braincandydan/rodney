# Web research

Use this skill when the user wants grounded answers from the web, source citation, and concise synthesis.

## Workflow

1. Clarify the question and success criteria.
2. Search / fetch sources as allowed by tools.
3. Summarize with citations; flag uncertainties.
4. **`remember`** key durable findings under `semantic` or `project` with tags.

## Output

- Bullet findings → implications → recommended next step.
